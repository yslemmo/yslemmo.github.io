NexT主题如何显示摘要

发表于 2019-02-03 | 更新于 2018-03-20 | 分类于 Hexo

刚创建好的网站，把几篇以前写的MarkDown文件放在_Post文件夹中，但是打开网站一下，发现在页面中每篇文章都全部显示了，翻看起来不方便。如何解决这个问题.

官方提供了三个解决办法!

  1. 在文章中使用手动截断的方法，截断的语句是<!-- more -->
  2. 在文章的 front-matter 中添加 description，并提供文章摘录
  3. 自动形成摘要，在主题配置文件中添加，默认截取的长度为 150字符，可以根据需要自行设定：
