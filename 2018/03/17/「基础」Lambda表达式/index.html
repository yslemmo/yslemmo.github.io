Lambda表达式

发表于 2018-03-17 | 更新于 2018-03-23 | 分类于 Unity

1. Lambda表达式概念

官方概念

Lambda 表达式是一种可用于创建 委托 或 表达式目录树 类型的 匿名函数 。 通过使用 lambda 表达式，可以写入可作为参数传递或作为函数调用值返回的本地函数。

  1. Lambda 表达式，是一种代码的语法格式；
  2. Lambda 表达式是对匿名方法语法格式的进一步简化；
  3. Lambda 表达式可以理解成是匿名方法的升级版。

【方法的三种表现形式】

普通方法–> 匿名方法–> Lambda 表达式
越往后越抽象，越往后理解成本也就越高。

2. Lambda表达式的语法

  1. 语法格式

委托类型变量= （[参数列表]）=> { 方法体; };

  1. 语法格式分析
  • => 是Lambda 表达式特有符号，读音是：goes to
  1. 如果没有参数列表，（）也不能省略，如：（）=>
  2. 如果参数列表中只有一个参数，（）可以省略
  3. 如果方法体内只有一句代码，可以省略代码块符号，也就是不用写{ }
