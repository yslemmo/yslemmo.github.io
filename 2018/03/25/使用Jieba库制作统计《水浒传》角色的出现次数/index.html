使用Jieba库制作统计《水浒传》角色的出现次数

发表于 2018-03-25 | 分类于 Python ， Module

这是学习Python一个星期依赖独立制作的一个比较小的案例脚本，计算一段文本的「词频」

案例介绍

案例准备
